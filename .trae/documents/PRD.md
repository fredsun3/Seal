## 1. 产品概述

本项目是一个以"篆书之美"为主题的汉字文化展示网站,聚焦于三种最具代表性的篆体——**李斯小篆**、**大篆(金文)**、**鸟虫篆**,以《百家姓》常用汉字为载体,通过精美的图文呈现,让用户直观感受汉字从甲骨金文到秦代规范小篆、再到装饰性鸟虫篆的演变之美。

- 主要目的:以视觉化、艺术化的方式展示三种篆体的字形差异与文化内涵,服务于汉字爱好者、书法学习者及传统文化传播场景
- 目标用户:书法爱好者、汉字文化研究者、中小学语文/书法教师、传统文化爱好者
- 市场价值:填补当前网络中"篆体对比展示"类高质量视觉产品的空白,具备教育与文化IP双重价值

## 2. 核心功能

### 2.1 功能模块
1. **首页(总览)**:三种篆体总览展示、文化背景导引、导航入口
2. **篆体详情页**:对应篆体的百家姓汉字展示、字形解析、历史背景

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 首页 | Hero 主视觉 | 三种篆体并排展示"篆"字,动态切换,文化标语 |
| 首页 | 篆体导航卡 | 三张卡片分别进入三种篆体详情,含字形预览与简介 |
| 首页 | 汉字演变时间轴 | 大篆→小篆→鸟虫篆的时间轴可视化 |
| 首页 | 文化背景 | 简述三种篆体的历史地位与艺术特征 |
| 小篆详情页 | 字形展示 | 李斯小篆版百家姓(赵钱孙李等)网格展示,悬停放大 |
| 小篆详情页 | 字形解析 | 选中汉字显示楷书对照、释义、字形结构说明 |
| 小篆详情页 | 历史背景 | 秦统一文字、李斯与《仓颉篇》介绍 |
| 大篆详情页 | 字形展示 | 金文版百家姓网格展示,带青铜器质感背景 |
| 大篆详情页 | 字形解析 | 选中汉字显示楷书对照、释义、金文拓片参考 |
| 大篆详情页 | 历史背景 | 商周青铜器铭文、毛公鼎、散氏盘介绍 |
| 鸟虫篆详情页 | 字形展示 | 鸟虫篆版百家姓网格展示,装饰性边框 |
| 鸟虫篆详情页 | 字形解析 | 选中汉字显示楷书对照、释义、鸟虫装饰特征 |
| 鸟虫篆详情页 | 历史背景 | 越王勾践剑铭、楚系鸟虫书、汉代印章应用 |

## 3. 核心流程

用户进入首页后,通过 Hero 主视觉感受三种篆体的整体美感,随后可通过导航卡片选择感兴趣的篆体进入详情页。在详情页中,用户浏览百家姓汉字网格,点击任一汉字查看楷书对照、释义与字形解析,并可切换不同篆体进行对比。

    "浏览百家姓字形网格" --> "点击单个汉字"
    "点击单个汉字" --> "查看楷书对照与释义"
    "查看楷书对照与释义" --> "切换其他篆体对比"

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**:墨黑(#1a1410)、宣纸米(#f4ecd8)、朱砂红(#c8392e)、古铜金(#b8893a)、青铜绿(#5a6b4a)
- **辅助色**:印章红(#a02818)、拓片褐(#8b6f47)
- **按钮风格**:仿印章方形按钮,朱砂底白字,带轻微浮雕阴影
- **字体**:
  - 标题:思源宋体(Noto Serif SC)粗体 + 仿篆体装饰
  - 正文:思源宋体常规
  - 篆体字形:使用 SVG 路径或图片资源呈现
- **布局风格**:卡片式 + 网格化,顶部固定导航,详情页采用左右分栏(字形展示 + 解析)
- **图标/装饰**:传统纹样(回纹、云纹、鸟虫纹)、印章元素、毛笔笔触

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|-----------|-------------|-------------|
| 首页 | Hero 主视觉 | 深墨色背景,中央三篆并排,朱砂印章式标题,淡入动画 |
| 首页 | 篆体导航卡 | 三栏卡片,古铜金边框,悬停浮起,字形预览 |
| 首页 | 时间轴 | 横向时间轴,青铜绿节点,滚动触发动画 |
| 小篆详情页 | 字形网格 | 6列网格,宣纸底纹,朱砂边框,悬停放大旋转 |
| 小篆详情页 | 解析面板 | 右侧抽屉式,楷书对照大字,释义文本 |
| 大篆详情页 | 字形网格 | 青铜器质感背景,金文字形带斑驳效果 |
| 鸟虫篆详情页 | 字形网格 | 装饰性鸟虫纹边框,字形带流动光效 |

### 4.3 响应式
- 桌面优先设计(1280px+ 完整体验)
- 平板(768-1280px):网格调整为 4 列,解析面板改为底部弹出
- 移动端(<768px):网格调整为 2 列,导航改为汉堡菜单,Hero 简化

### 4.4 视觉细节
- 全站背景:宣纸纹理 + 微噪点,营造古籍质感
- 装饰元素:页面四角回纹边框,标题旁印章式落款
- 动效:页面加载时篆体字形逐字浮现,悬停时字形轻微旋转放大,滚动时时间轴节点依次点亮
